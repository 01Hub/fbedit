Modifying project groups

There are times when you wish to better organize the files in the Project box. This is possible using the groups editor from the Project menu. Select Project Groups from the Project menu and this dialog will appear:

You can add a group by typing it's name in the edit control and clicking Add Group. Change a goroup name by selecting the group in the right grid control and type in the new name. To move files to a group, select the group in the right grid control and double click the file you wish to move in the tree view to the left.
