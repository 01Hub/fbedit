Welcome to Fiview
-----------------

Fiview has a help-text built into it, and this can be viewed either in
the app itself, or on the command line.  To get started, try one of
these:

To see usage instructions:

  fiview

To see helptext on the command-line:

  fiview -h

To bring up a demo set of filters and see the help-text within the app:

  fiview -D

To try a simple filter:

  fiview -i LpBe4/0.01
