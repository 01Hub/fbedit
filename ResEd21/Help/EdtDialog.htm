Dialog Editor



 Dialog editor.

 

To put a control on your dialog select a control from the toolbox to the left. Then left click on the dialog where you want your control. You can also hold the button and draw it to the size you want.
NOTE!
If you hold the Ctrl key while selecting from the toolbox the selected button will stay down. This can save time when you want several controls of the same type.
You select an existing control by left clicking on the control. If you hold the Ctrl key you can select more than one control. You can also left click on the dialog and hold the mouse button and draw a rectangle. All controls in the rectangle will be selected when you release the button.
