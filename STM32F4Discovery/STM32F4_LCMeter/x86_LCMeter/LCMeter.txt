Features:
-------------------------------------------------------------------------------
LC Meter
High speed clock
Scope
DDS Waveform generator
Logic analyser

-------------------------------------------------------------------------------
Port pins used:
-------------------------------------------------------------------------------
PA.00         	Frequency counter input (TIM5)
PA.01         	Frequency counter input (TIM2)
PA.04         	Scope V-Pos DAC1 Output
PB.07         	High Speed Clock Output
PB.13         	SPI SCK
PB.15         	SPI MOSI
PC.01         	Scope ADC Input
PD.00         	Frequency counter select0 Output
PD.01         	Frequency counter select1 Output
PD.02         	Frequency counter select2 Output
PD.06         	LCMeter L/C selection Output, C = Low L = High
PD.07         	LCMeter calibration Output
PD.08         	USART3 TX Output
PD.09         	USART3 RX Input
PE.03			Scope X5 Output OD
PE.04			Scope X10 Output OD
PE.05			Scope X10 Output OD
PE.08 to PE.15	LGA Inputs
